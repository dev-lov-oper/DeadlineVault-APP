# 🚀 DeadlineVault  
### 📱 Smart Deadline Manager for Students (Android App)

> *“Don’t just track deadlines — beat them.”*

---

## 📌 Overview  
DeadlineVault is an intelligent Android application designed to help students manage academic deadlines efficiently.

It uses a **priority-based system** to automatically rank tasks based on urgency, ensuring you always focus on what matters most.

---

## ✨ Features  

### 🔥 Intelligent Priority Queue  
- Tasks are automatically sorted based on remaining time  
- Most urgent tasks appear first  
- No manual prioritization needed  

---

### 💾 Offline Storage (SQLite)  
- Uses SQLite database for local storage  
- Works completely offline  
- Fast and lightweight  

---

### 🎨 Color-Based Urgency System  
- 🔴 Red → Urgent (Due soon)  
- 🟡 Yellow → Moderate  
- 🟢 Green → Safe  

---

### ⏳ Real-Time Updates  
- Deadlines update dynamically  
- Task priorities change automatically over time  

---

### 📱 Clean UI  
- Simple and user-friendly design  
- Built for productivity  

---

## 🛠️ Tech Stack  

- Language: Java / Kotlin  
- Platform: Android SDK  
- Database: SQLite  
- IDE: Android Studio  
- Build System: Gradle  

---

## 🧩 How It Works  

1. User adds a task with a deadline  
2. Task is stored in SQLite database  
3. App calculates remaining time  
4. Tasks are sorted using priority logic  
5. UI displays tasks based on urgency
